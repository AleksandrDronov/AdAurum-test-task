### Запуск проекта:

1. Клонировать репозиторий.

   git clone git@github.com:AleksandrDronov/AdAurum-test-task.git

2. Войти в папку проекта.

   cd AdAurum-test-task

3. Установить зависимости.

   npm install

4. Запуск.

   npm run dev
